# Learn JavaScript with p5.js

Learn JavaScript fundamentals through creative coding.

## Quick Start

```bash
# From root directory
npm run learn

# Or from this directory
npm run dev
```

**URL:** http://localhost:5176

## Chapters

### Chapter 01: Hello p5.js
- **01-first-sketch** ✨ - Animated circles following mouse (working example)
- **02-drawing-shapes** - Basic shape drawing
- **03-color-intro** - Introduction to color

### Chapter 02: JavaScript Basics
- **01-variables** - Variables and data types
- **02-functions** - Creating and using functions
- **03-conditionals** - If statements and logic

### Chapter 03: Color and Drawing
- **01-color-basics** - Color fundamentals
- **02-rgb-hsl** - RGB and HSL color modes
- **03-transparency** - Alpha and transparency

## Working Example

Chapter 01 includes a complete working example demonstrating:
- ✅ Canvas setup
- ✅ Animation loop
- ✅ Mouse tracking with easing
- ✅ Multiple animated shapes
- ✅ Interactive controls
- ✅ Console logging for debugging

```bash
npm run learn
```

## Creating New Sketches

```bash
# Copy template
cp -r ../../../templates/global-mode src/chapters/01-hello-p5js/04-my-sketch

# Edit sketch
nano src/chapters/01-hello-p5js/04-my-sketch/sketch.js
```

## Tips

- **Learn by doing**: Type out examples, don't just copy
- **Experiment**: Modify values and see what happens
- **Debug with console**: Use `console.log()` liberally
- **Save your work**: Use `saveCanvas()` to export
